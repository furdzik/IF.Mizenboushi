## Getting started

```
npm install --save-dev eslint stylelint-processor-styled-components @furdzik/stylelint
```

---

`.stylelintrc`

```javascript
  {
    "processors": [
      "stylelint-processor-styled-components"
    ],
      "extends": [
      "@furdzik/stylelint"
    ]
  }
```
